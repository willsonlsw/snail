直流电机智能调速系统

本人大二与电子系的同学合作参与学校的“百项工程”时开发的系统，我负责软件部分的开发。

该系统基于单片机，能较为精确地调节直流电机的转速，且具有与电机无关的特性，并且能自动判断电机的运行状态与负载而调节电机的转速。
系统集成了三种传感器：
1.光电传感器，用于测电机的转速
2.加速度传感器，用于测电机的运行状态
3.红外测距传感器，用于障碍检测

由于单片机的速度确实很慢，故起名Snail
